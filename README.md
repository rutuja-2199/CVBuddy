# CVBuddy
Create resumes that get hired.

## Description
CVBuddy is a web application that helps users create professional CVs/resumes easily. Inspired by FlowCV, it offers multiple demo styles for users to choose from. Users can select a style, fill in their details through a simple form, and generate a CV in the chosen style. The app uses Bootstrap for responsive design and allows downloading the CV as a PDF.

## Features
- Multiple CV style options
- User-friendly form for entering details
- Live preview of the generated CV
- PDF download functionality
- Responsive design with Bootstrap

## How to Use
1. Open `index.html` in a web browser.
2. Select a CV style from the available options.
3. Fill in your personal and professional details in the form.
4. Click "Generate CV" to see the preview.
5. Download the CV as a PDF.
